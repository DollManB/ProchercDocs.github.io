# Говори мой друг

Игрок может разговарить, ну точнее выводить <span style={{color: '#ecf756'}}>сообщение</span> в чат

example:

```js
Player.sendMessage("text{STRING}");
```

с переменной:
```js
const text = "hi";

Player.sendMessage(text);
```

про Player можете узнать [тут](../PlayerAPI/entered)
