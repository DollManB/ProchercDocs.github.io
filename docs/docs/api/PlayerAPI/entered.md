# Player API

Player API предоставляет полный набор инструментов для работы с игроками в игре. Этот API позволяет управлять всеми аспектами игрока: от базовой информации до сложных взаимодействий.

## Лучшие практики
1. **Обработка ошибок**: Проверяйте возвращаемые значения на null/undefined

## Ограничения

- Некоторые методы могут не работать с не-игровыми сущностями
- Модификации инвентаря могут конфликтовать с другими модами
- Физические изменения могут не синхронизироваться в реальном времени в мультиплеере
