# Введение в движок

Добро пожаловать в документацию нашего движка!

## Описание

Этот мод позволяет вам создавать свои творения прямо в ***майнкрафте***

## Установка

1. Установите [Fabric API](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
2. Скачайте последний релиз нашего мода с [Modrinth](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTcBHWEvGm26FCRTCGZ9qFB7GqZLfJY5ec40g&s) либо с [CurseForge](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTcBHWEvGm26FCRTCGZ9qFB7GqZLfJY5ec40g&s)
3. Поместите файл .jar в папку mods
4. Запустите игру с Fabric loader
