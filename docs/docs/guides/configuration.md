# Конфигурация мода

## Файл конфигурации

После первого запуска мода будет создан файл конфигурации:
`config/yourmod.json`

## Основные настройки

```json
{
  "enabled": true,
  "debugMode": false,
  "featureX": {
    "enable": true,
    "value": 10
  },
  "featureY": {
    "enable": false,
    "cooldown": 30
  }
}
```

## Параметры конфигурации

| Параметр | Тип | Описание | Значение по умолчанию |
|----------|-----|-----------|-----------------------|
| enabled | boolean | Включить/отключить мод | true |
| debugMode | boolean | Включить отладочные сообщения | false |
| featureX.enable | boolean | Включить функцию X | true |
| featureX.value | number | Значение для функции X | 10 |
| featureY.enable | boolean | Включить функцию Y | false |
| featureY.cooldown | number | Задержка для функции Y (секунды) | 30 |

## Команды в игре

| Команда | Описание |
|---------|----------|
| `/yourmod reload` | Перезагрузить конфигурацию |
| `/yourmod status` | Показать статус мода |
| `/yourmod help` | Показать эту справку |

## Перезагрузка конфигурации

Изменения в конфигурации применяются:
1. Автоматически при перезагрузке мира
2. Вручную с помощью команды `/yourmod reload`
3. При перезапуске игры
