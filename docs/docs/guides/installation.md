# Установка мода

## Системные требования
- Minecraft 1.20.4+
- Fabric Loader 0.14.22+
- Fabric API 0.90.0+

## Шаги установки

### 1. Установка Fabric
1. Скачайте Fabric Installer с [официального сайта](https://fabricmc.net/use/)
2. Запустите installer и выберите версию Minecraft
3. Нажмите "Install" и дождитесь завершения

### 2. Установка Fabric API
1. Скачайте Fabric API с [CurseForge](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
2. Поместите файл .jar в папку `mods` вашего Minecraft профиля

### 3. Установка нашего мода
1. Скачайте последний релиз мода с [GitHub](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcTcBHWEvGm26FCRTCGZ9qFB7GqZLfJY5ec40g&s)
2. Поместите файл .jar в папку `mods`
3. Запустите игру с Fabric профилем

## Проверка установки
После запуска игры вы должны увидеть сообщение в чате:
`[YourMod] Successfully loaded version X.X.X`

Если сообщение не появилось, проверьте логи игры в `.minecraft/logs/latest.log`
