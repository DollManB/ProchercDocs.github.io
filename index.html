Player API
Player API предоставляет полный набор инструментов для работы с игроками в игре. Этот API позволяет управлять всеми аспектами игрока: от базовой информации до сложных взаимодействий.
Лучшие практики

Обработка ошибок

Проверяйте возвращаемые значения на null/undefined

Ограничения
  • Некоторые методы могут не работать с не-игровыми сущностями
  • Модификации инвентаря могут конфликтовать с другими модами
  • Физические изменения могут не синхронизироваться в реальном времени в мультиплеере
Методы API
getName()
Получить имя игрока
let name = Player.getName();
// пример использования
let name = Player.getName();
Player.sendMessage(name);
sendMessage(text)
Игрок может разговаривать, выводить сообщение в чат
Player.sendMessage("text{STRING}");

// с переменной:
const text = "hi";
Player.sendMessage(text);
getServerPlayer()
Получить ServerPlayerEntity - компонент игрока зависящий от net.minecraft.server.network

Полную информацию можно посмотреть тут

let player = Player.getServerPlayer(); // возвращает ServerPlayerEntity в player

Server API
Server API предоставляет мощные инструменты для управления сервером и глобальными игровыми процессами. Этот API позволяет контролировать все аспекты серверной части игры, от управления игроками до выполнения команд и модификации игрового мира.
Методы API
executeCommand(command)
Этот метод позволяет от имени сервера выполнить команду
Необязательно писать через [ / ]
Server.executeCommand("example command");

// пример:
Server.executeCommand("time set day");

// еще пример:
let plr = Server.getRandomPlayer();
let name = plr.getName();
Server.executeCommand("give " + name + " minecraft:stick");
getOnlinePlayer()
Получить список игроков (имена)
// пример:
let list = Server.getOnlinePlayer();
getPlayerByName(nickname)
Получить игрока по имени
// пример:
let player = Server.getPlayerByName("Nickname"); // передает PlayerAPI
// после этого теперь player у нас является зависимым от PlayerAPI
getRandomPlayer()
Получить рандомного игрока
// пример:
let player = Server.getRandomPlayer(); // передает PlayerAPI
// теперь player у нас становиться зависимым от PlayerAPI
broadcast(message)
Отправляет сообщение в чат от имени сервера
// пример:
Server.broadcast("привет мир!");
Полный пример использования
// Пример: Выдать рандомному игроку алмаз
function giveRandomPlayerDiamond() {
  let randomPlayer = Server.getRandomPlayer();
  if (randomPlayer) {
    let playerName = randomPlayer.getName();
    Server.executeCommand("give " + playerName + " minecraft:diamond");
    Server.broadcast("Игрок " + playerName + " получил алмаз!");
  }
}
