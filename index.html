⚡ ProchercEngine

Мощный скриптовый движок для Minecraft на основе Rhino JavaScript. Создавайте катсцены, NPC, кастомные предметы и интерактивные механики без компиляции модов.

Установка

1

Установите мод

Поместите файл мода в папку mods/

2

Создайте папку скриптов

Скрипты хранятся в config/ProchercEngine/scripts/

3

Создайте первый скрипт

Файлы скриптов имеют расширение .pec

Команды движка:
  • /prochercengine run-script <name> - Запустить скрипт
  • /prochercengine stop-script <name> - Остановить скрипт
  • /prochercengine list - Список скриптов
  • /prochercengine list-running - Запущенные скрипты
  • /prochercengine start-player <player> <script> - Запустить для игрока

Первый скрипт

hello.pec 
// Отправить сообщение игроку
Player.sendMessage("§aПривет, " + Player.getName() + "!");

// Телепортировать игрока
Player.teleport(0, 64, 0);

// Установить блок
World.setBlock(0, 63, 0, "minecraft:diamond_block");

// Воспроизвести звук
Sound.playForPlayer("minecraft:entity.experience_orb.pickup");

API Player

Управление игроком, инвентарём, моделями и скинами

sendMessage (text: String)

Отправить сообщение игроку с поддержкой цветовых кодов

Player.sendMessage("§aУспех! §7Это зелёное сообщение");
teleport (x: double, y: double, z: double)

Телепортировать игрока в указанные координаты

Player.teleport(100, 64, -200);
getPos () → Vec3d

Получить текущую позицию игрока

var pos = Player.getPos();
Player.sendMessage("X: " + pos.x + ", Y: " + pos.y + ", Z: " + pos.z);
getName () → String

Получить имя игрока

hasItem (itemId: String, count: int) → boolean

Проверить наличие предмета в инвентаре

if (Player.hasItem("minecraft:diamond", 5)) {
    Player.sendMessage("§aУ вас есть 5 алмазов!");
}
getItemCount (itemId: String) → int

Получить количество предмета в инвентаре

var diamonds = Player.getItemCount("minecraft:diamond");
Player.sendMessage("Алмазов: " + diamonds);
removeItem (itemId: String, count: int) → boolean

Удалить предмет из инвентаря

if (Player.removeItem("minecraft:diamond", 1)) {
    Player.sendMessage("§eАлмаз удалён из инвентаря");
}
setModel (modelPath: String, animationPath: String, texturePath: String)

Установить кастомную модель игроку (GeckoLib формат)

Player.setModel(
    "geo/custom_player.geo.json",
    "animations/custom_player.json",
    "textures/custom/player.png"
);
applySkinAsTexture (nick: String)

Применить скин другого игрока как текстуру модели

Player.applySkinAsTexture("Notch");
resetModel ()

Сбросить кастомную модель игрока

input (prompt: String, callback: Function)

Получить ввод от игрока через чат

Player.input("Введите имя:", function(result) {
    Player.sendMessage("Вы ввели: " + result);
});

API World

Управление блоками и миром

setBlock (x: int, y: int, z: int, blockId: String)

Установить блок в мире

World.setBlock(0, 64, 0, "minecraft:diamond_block");
getBlock (x: int, y: int, z: int) → String

Получить ID блока по координатам

var block = World.getBlock(0, 64, 0);
// Возвращает "minecraft:diamond_block"
getBlock (x: int, y: int, z: int, blockId: String) → boolean

Проверить, является ли блок указанным

if (World.getBlock(0, 64, 0, "minecraft:lever")) {
    Player.sendMessage("Это рычаг!");
}
spawnExplosion (x: double, y: double, z: double, power: float)

Создать взрыв в указанной позиции

World.spawnExplosion(100, 64, 100, 4.0);

API Entity

Создание и управление NPC с кастомными моделями

create (pos: Vec3d, name: String, modelPath: String, animationPath: String, texturePath: String, fraction?: String) → EntityHandle

Создать NPC с кастомной моделью

var npc = Entity.create(
    pos(100, 64, 200),
    "Торговец",
    "geo/npc.geo.json",
    "animations/npc.json",
    "textures/npc.png",
    "Neutral"
);

EntityHandle методы

move (pos: Vec3d, speed?: double)

Заставить NPC идти к точке

npc.move(pos(110, 64, 200), 1.5);
look (pos: Vec3d)

Разово посмотреть на точку

lookAlways (pos: Vec3d | PlayerAPI)

Постоянно смотреть на точку или игрока

npc.lookAlways(Player);
// или
npc.lookAlways(pos(100, 65, 200));
setFraction (fraction: String)

Установить поведение NPC

Фракции:
  • "Neutral" - мирный, но ответит ударом
  • "Aggressive" - нападает при виде игрока
  • "Peaceful" - никогда не атакует
playAnimation (name: String, mode: String)

Воспроизвести анимацию

npc.playAnimation("wave", "loop");
// mode: "loop", "once", "hold"
say (text: String)

Показать текст над NPC

npc.say("Привет, путник!");
kill ()

Убить NPC

remove ()

Удалить NPC из мира

API Item

Создание кастомных предметов

create (name: String, modelPath: String, displayPath: String, animationPath: String, texturePath?: String) → ItemHandle

Создать кастомный предмет

var sword = Item.create(
    "Меч Дракона",
    "geo/dragon_sword.geo.json",
    "display/dragon_sword.json",
    "animations/dragon_sword.json",
    "textures/items/dragon_sword.png"
);

ItemHandle методы

give (playerName: String | PlayerAPI)

Выдать предмет игроку

sword.give(Player);
// или
sword.give("Steve");
spawn (pos: Vec3d)

Спавнить предмет в мире

sword.spawn(pos(100, 65, 200));

API Camera

Создание катсцен и управление камерой

play (fileName: String, durationSeconds: double, player: PlayerAPI)

Воспроизвести катсцену из файла

Camera.play("intro_cutscene.cadr", 10.0, Player);
stop (player: PlayerAPI)

Остановить катсцену

isPlaying (player: PlayerAPI) → boolean

Проверить, воспроизводится ли катсцена

Создание катсцены:
  1. Возьмите предмет "Камера" в руку
  2. Нажмите ПКМ для добавления точек
  3. Используйте /procherc_camera save yes для сохранения

API Async

Асинхронные операции и задержки

await (seconds: double)

Приостановить выполнение скрипта

Player.sendMessage("Подождите 3 секунды...");
Async.await(3);
Player.sendMessage("Готово!");
await (condition: Function, callback: Function)

Ждать выполнения условия, затем вызвать callback

Async.await(function() {
    return World.getBlock(0, 64, 0, "minecraft:lever");
}, function() {
    Player.sendMessage("§aРычаг активирован!");
});

API Event

Обработка событий игрока

onBlockBreak (player: ServerPlayerEntity, callback: Function)

Вызывается при ломании блока

Event.onBlockBreak(Player.getServerPlayer(), function(e) {
    Player.sendMessage("Вы сломали: " + e.block);
});
onBlockPlace (player: ServerPlayerEntity, callback: Function)

Вызывается при установке блока

onJump (player: ServerPlayerEntity, callback: Function)

Вызывается при прыжке

onCrouch (player: ServerPlayerEntity, callback: Function)

Вызывается при приседании

onSprint (player: ServerPlayerEntity, callback: Function)

Вызывается при беге

onDeath (player: ServerPlayerEntity, callback: Function)

Вызывается при смерти

onDamage (player: ServerPlayerEntity, callback: Function)

Вызывается при получении урона

Event.onDamage(Player.getServerPlayer(), function(e) {
    Player.sendMessage("Урон: " + e.amount);
});
onAttack (player: ServerPlayerEntity, callback: Function)

Вызывается при атаке существа

onSwim (player: ServerPlayerEntity, callback: Function)

Вызывается при плавании

API Sound

Воспроизведение звуков

playForPlayer (player: PlayerAPI, soundId: String, volume?: float, pitch?: float)

Воспроизвести звук для конкретного игрока

Sound.playForPlayer(Player, "minecraft:entity.experience_orb.pickup", 1.0, 1.0);
playAt (pos: Vec3d, soundId: String, volume?: float, pitch?: float)

Воспроизвести звук в позиции

Sound.playAt(pos(100, 64, 200), "minecraft:block.anvil.use", 1.0, 1.0);
playAll (soundId: String, volume?: float, pitch?: float)

Воспроизвести звук для всех игроков

stopForPlayer (player: PlayerAPI, soundId: String)

Остановить звук для игрока

stopAllForPlayer (player: PlayerAPI)

Остановить все звуки для игрока

API Particle

Bedrock-стиль частицы (Snowstorm формат)

Start (effectId: String, pos: Vec3d)

Запустить эффект частиц для всех игроков

Particle.Start("snowstorm:rain", pos(100, 64, 200));
Start (effectId: String, pos: Vec3d, player: PlayerAPI)

Запустить эффект частиц только для указанного игрока

Stop (pos: Vec3d)

Остановить все частицы в позиции

StopAll (effectId?: String)

Остановить все частицы указанного типа или все вообще

Particle.StopAll("snowstorm:rain");
Particle.StopAll(); // Все частицы
Файлы частиц:

Поместите .particle.json файлы в:

resourcepacks/<Pack>/assets/<namespace>/particles/<name>.particle.json

API Gui

Создание кастомных интерфейсов

Создать GUI

{}

API Hitbox

Интерактивные зоны и триггеры

create (x: double, y: double, z: double, width: float, height: float) → HitBoxEntity

Создать интерактивную зону

var zone = Hitbox.create(100, 64, 200, 3.0, 2.0);
create (name: String, x: double, y: double, z: double, width: float, height: float) → HitBoxEntity

Создать именованную зону

onInteract (entity: InteractionEntity, callback: Function)

Вызывается при взаимодействии с зоной

Hitbox.onInteract(zone, function(e) {
    e.player.sendMessage("Вы нажали на зону!");
});
onEnter (hitbox: Entity, callback: Function)

Вызывается при входе игрока в зону

onExit (hitbox: Entity, callback: Function)

Вызывается при выходе игрока из зоны

Hitbox.onEnter(zone, function(e) {
    e.player.sendMessage("Добро пожаловать!");
});

Hitbox.onExit(zone, function(e) {
    e.player.sendMessage("До свидания!");
});
remove (entity: InteractionEntity)

Удалить зону

Java интеграция

ProchercEngine предоставляет прямой доступ к Java классам Minecraft через функцию Java().

Примеры использования 
// Получить класс ItemStack
var ItemStack = Java("net.minecraft.item.ItemStack");

// Использовать алиасы
var Items = Java("Items");
var stack = new ItemStack(Items.DIAMOND_SWORD, 1);

// Доступ к NBT
var NbtCompound = Java("NbtCompound");
var nbt = new NbtCompound();
nbt.putString("key", "value");

// Математические функции
var Math = Java("Math");
var random = Math.random();

Примеры скриптов

Простой диалог с NPC

// Создаём NPC
var npc = Entity.create(
    pos(100, 64, 200),
    "Старик",
    "geo/villager.geo.json",
    "animations/villager.json",
    "textures/villager.png"
);

// NPC смотрит на игрока
npc.lookAlways(Player);

// Создаём зону взаимодействия
var zone = Hitbox.create(100, 64, 200, 2, 3);
Hitbox.onInteract(zone, function(e) {
    npc.say("Привет, путник!");
    Async.await(2);
    npc.say("Добро пожаловать в нашу деревню!");
});

Проверка предметов

// Проверяем есть ли у игрока алмазы
if (Player.hasItem("minecraft:diamond", 5)) {
    // Забираем алмазы
    Player.removeItem("minecraft:diamond", 5);
    
    // Даём награду
    var sword = Item.create(
        "Алмазный меч",
        "geo/diamond_sword.geo.json",
        "display/sword.json",
        "animations/sword.json"
    );
    sword.give(Player);
    
    Player.sendMessage("§aВы обменяли 5 алмазов на меч!");
} else {
    Player.sendMessage("§cНужно 5 алмазов для обмена!");
}

События игрока

// Реакция на прыжок
Event.onJump(Player.getServerPlayer(), function(e) {
    Particle.Start("snowstorm:jump", pos(e.x, e.y, e.z));
    Sound.playAt(pos(e.x, e.y, e.z), "minecraft:entity.phantom.flap", 0.5, 1.5);
});

// Реакция на смерть
Event.onDeath(Player.getServerPlayer(), function(e) {
    Server.broadcast("§c" + Player.getName() + " погиб!");
});

// Реакция на урон
Event.onDamage(Player.getServerPlayer(), function(e) {
    Player.sendMessage("§cУрон: " + e.amount);
});
